Box Shadow

box-shadow is a nice way to add more layers of same shape.
Syntax:
box-shadow: h-offset v-offset blur spread color

As an example, we will create smaller clones of previous shapes with using box-shadow property:

the div
div {
	width: 100px;
	height: 100px;
	display: inline-block;
	background-color: lightgray;
	position: absolute;
	top: calc(50% - 50px);
	left: calc(50% - 50px);
	box-shadow: 0px 100px 0 -20px gray;
}
::before {
	content: "::before";
	width: 100px;
	height: 100px;
	display: inline-block;
	background-color: palegreen;
	position: absolute;
	left: -100%;
	border-radius: 100%;
	box-shadow: 0px 100px 0 -20px green;
}



::after {
	content: "::after";
	width: 0px;
	height: 0px;
	display: inline-block;
	background-color: transparent;
	position: absolute;
	left: 100%;
	border: 50px solid transparent;
	border-right: 50px solid lightskyblue;
	border-bottom: 50px solid lightskyblue;
	box-shadow: 0px 100px 0 -20px blue;
}

Finding! Trianlges are not working with box-shadow!

On the other hand, we can have more shadows, thus more clones:

the div
div {
	box-shadow: 
		0px 100px 0 -20px gray,
		0px 100px 0 -10px dimgray;
}

::before {
	box-shadow: 
		0px 100px 0 -20px green,
		0px 100px 0 -10px darkgreen;
}

::after {
	box-shadow: 
		0px 100px 0 -20px blue,
		0px 100px 0 -10px darkblue;
}

The size of box-shadow of the (pseudo) element is calculated as:
shadow width = width + left border + right border + spread
shadow height = height + top border + bottom border + spread

Drop Shadow

One solution to our shadow problem with Triangle is filter: drop-shadow(...);
Syntax:
filter: drop-shadow(horizontal-shadow vertical-shadow blur color);

the div
div {
	filter: drop-shadow(0px -100px 0 gray);
}

::before {
	filter: drop-shadow(0px 100px 0 goldenrod);
}


A single drop-shadow is applied to all (pseudo) elements, so be carefull.
In next example we remove previous style and apply drop-shadow only for our triangle element:

the div
::after {
	filter: drop-shadow(0px 50px 0 burlywood);
}

This is what happends when we have more drop-shadow in filter:

the div
::after {
	filter: drop-shadow(0px 50px 0 burlywood)
		drop-shadow(-25px -50px 0 goldenrod)
		drop-shadow(-75px 0px 0 orangered);
}

Drop Shadow + Box Shadow

It is also important to know how these two work together, as both of them can create more clones (aka shadows) of our (pseudo) elements.
Box Shadow can be scaled up and down, and cannot be used with triangles.
Drop Shadow cannot be scalled, works with triangles (or any shape) and tries to apply on everything as much as possible.
Both of them add new layers bellow previous ones, so order matters. Let's try both of them on our circle element

the div
::before {
	filter: drop-shadow(0px 50px 0 burlywood);
	box-shadow: 175px 0px 0 -20px green;
}

Filter drop-shadow is applied after box-shadow, and applied to everything (element + his box shadows).
Box Shadow will add +1 'shade', Drop Shadow will add +N 'shades'.

Gradients - Linear gradients

If you don't have a clue about CSS Gradients, please look here before reading further.
This part is an extension to it, so quickly chech basics and come back.

the div
Back to our original div.
It has these starting CSS:


div {
	width: 100px;
	height: 100px;
	display: inline-block;
	background-color: lightgray;
	position: absolute;
	top: calc(50% - 50px);
	left: calc(50% - 50px);
}

Let's make some basic triangles:

the div
div {
	background-image: 	
		linear-gradient(to top right, transparent, transparent 70%, gray 70%),
		linear-gradient(to bottom right, transparent, transparent 60%, black 60%),
		linear-gradient(to top right, aliceblue, aliceblue 50%, transparent 50%);
}

The trick to have sharp edges is achieved by same % values for both colors

As you can see, the first triangle is on top, and every next one is under previous one. Same logic as with shadows.
Similarly, you can draw lines:

the div
div {
	background-image: 	
		linear-gradient(
			transparent, transparent 31%, black 31%, black 33%, transparent 33%, transparent 66%, red 66%, red 68%, transparent 68%
		), 
		linear-gradient(
			to right, transparent, transparent 31%, green 31%, green 33%, transparent 33%, transparent 66%, blue 66%, blue 68%, transparent 68%
		);
}

The problem is that you will always have line from one end to another.
Common solutions are placing an overlapping gradient before it, and having another element in front of it.
Another fancy thing that people do with line gradients are stripes.

the div
div {
	background-image: repeating-linear-gradient(to bottom right, aliceblue, aliceblue 5%, transparent 5%, transparent 10%);
}
